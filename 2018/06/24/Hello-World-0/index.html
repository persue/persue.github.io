二 文章 & 草稿

文章

1、新建文章

$ hexo new “new artical”

2、属性
包含如下几类：
layout Layout
title 文章的标题
date 创建日期
updated 修改日期
comments 是否开启评论
tags 标签
categories 分类
permalink url中的名字

3、分类

categories:

4、摘要：
同wordpress一样，之上的内容为摘要.

草稿

草稿可当作是一种“私密文章”功能。

$ hexo new draft “new draft”

将草稿变为文章：

$ hexo publish [layout]
